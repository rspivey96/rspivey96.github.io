Projects
CommBo - Digital Communication Device Spring 2018

 CommBo is a dynamic Augmentative and Alternative Communication (AAC) device, aimed at giving a user with speech impairment the most flexible app they've seen to date. The application is truly one of kind due to the customizable user interface, machine learned recommendations, and text to speech capabilities.
 Built by a team of five during a senior capstone design course, the motivation for the application comes from a family friend suffering from mental retardation. The family of this friend was in need of an application that could out-perform the physical picture board they've been currently using. Furthermore, other digital alternatives were found to be costly. CommBo offers ease of use through a portable design, capable of functionality regardless of the device. In addition to this, CommBo is currently free to use.

Relational Database Management System (RDBMS) Fall 2016

 Built by a group of three in a programming studio course (Fall 2016), this relational database management system was designed to be a sports management system. Although it could have any type of application, our system gave the user the ability to keep up with stats behind multiple teams, players, and venues. The three parts include: engine, parser, and interactive system. Built in Java, the parser is responsible for converting user-generated MySQL into the correct Java function calls within the engine. The engine contains the functionality of standard MySQL database. Lastly, the interactive system. provides a list of commands for the user and passes user commands to the parser.
